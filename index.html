Brevet AB Agent Skills
AI-Powered Tax Ecosystem

Otomatiskan Alur Kerja Perpajakan Brevet AB

Asisten AI pintar untuk konsultan dan profesional pajak. Dilengkapi kapabilitas rekonsiliasi fiskal, pemantau regulasi harian, hingga penanganan sengketa SP2DK.

GitHub Repository Cara Instalasi
🚨 Salin Prompt untuk Instalasi Cepat
Tolong unduh dan instal agent skills Brevet AB dari https://github.com/lensetek/BrevetAB-Agent-Skills dengan cara melakukan clone repositori tersebut, mengekstrak data skill dari folder `skills/`, lalu mendaftarkan path-nya ke dalam berkas `skills.json` di konfigurasi agent saya.

9 Spesialis Perpajakan AI

👑

Tax Orchestrator

Manajer utama yang menerima perintah pengguna, mengoordinasikan agen spesialis, dan merangkum laporan akhir.

🔍

Tax Regulation Monitor

Memantau aturan perpajakan terbaru secara harian (UU, PMK, PER) untuk meminimalkan risiko ketidakpatuhan.

📊

Accounting Data Processor

Memproses file Excel/CSV atau foto kuitansi (multimodal OCR) untuk eksekusi rekonsiliasi fiskal.

📝

Tax Report Generator

Menghitung pajak terutang dan menyusun draf formulir lampiran SPT Badan/OP siap lapor.

🛡️

Agent Update Manager

Memantau pembaruan Git, memindai kebocoran kredensial secara aman, dan meminta izin sebelum memperbarui.

🧠

User Personalization Learner

Menghafal profil Wajib Pajak klien, format pelaporan, toleransi risiko, dan belajar lewat perintah /learn.

⚖️

Withholding Tax & VAT Analyst

Menganalisis objek pemotongan/pemungutan PPh Pasal 21/22/23/26/4(2) dan ekualisasi e-Faktur PPN.

🏛️

Tax Dispute Defender

Menyusun draf surat tanggapan formal atas surat imbauan klarifikasi SP2DK atau teguran resmi DJP.

💡

Tax Planner & Strategist

Simulasi efisiensi beban pajak secara legal (seperti skema natura vs tunjangan uang tunai).
